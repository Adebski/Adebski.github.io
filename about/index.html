About me

Hello,

My name is Andrzej Debski, and I’m a software developer engineer, currently working for Amazon Alexa in Cambridge, UK.

Most of my experience comes from working on backend systems in JVM-based languages like Java and Scala but I’m also familiar with languages like Python, Bash, and JavaScript. I have some experience with front-end technologies but I don’t consider myself proficient in them. Actually, this is a first website that I’ve built from scratch.

My interests include:

  • distributed systems,
  • software craftsmanship,
  • functional programming,
  • mentoring and teaching others.

I’ve decided to start my own blog for two reasons:

  1. working on my writing skills,
  2. sharing my thoughts and observations on various topics related to software engineering with a wider audience.
